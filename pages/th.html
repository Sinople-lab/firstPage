Listas & tablas

Como hacer listas y tablas

Existen 3 tipos de listas en html: Ordered list, Unordered list & Description list; Ordered List se traduce como lista ordenada y se define con <ol>, este tipo de lista se caracteriza por tener sus elementos enumerados; Unordered list se define con <ul> y se caracteriza por no tener sus elementos enumerados; y por ultimo esta Description List, definida por <dl> la cual se usa para crear una lista descriptiva en la que los elementos se ordenan dentro de subcategorias. Para el caso de >ol> y <tu>, sus elementos se declaran con <li> y para <dl> se declaran con la etiqueta <dt>, que se refiere a Description Term y <dd>, que viene de Description Data (aunque esta es interpretacion mia ya que en W3Schools solo menciona que viene de description pero quise encontrarle mas sentido a la doble "d")

Para crear una tabla se utiliza la etiqueta <table>; dentro de esta se utiliza la etiqueta <tr>, que abrevia "table row" y con esta se crea una fila; dentro de <tr> se pueden colocar <th>, que abrevia "table head" y son los encabezados de las columnas; y por ultimo tenemos <td>, que define el contenido de la casilla y abrevia "table data".

Ejemplo de tabla

<table>
  <tr>
    <th>Nombre</th>
    <th>Apellido</th>
    <th>Edad</th>
  </tr>
  <tr>
    <th>Craig</th>
    <th>Reynols</th>
    <th>23</th>
  </tr>
  <tr>
    <th>Mark</th>
    <th>Schafer</th>
    <th>19</th>
  </tr>
  <tr>
    <th>Jacob</th>
    <th>Smith</th>
    <th>18s</th>
  </tr>
  </table>

Ejemplo de lista ordenada

<ol>
 <li>Mark</li>
 <li>Jacob</li>
 <li>Craig</li>
</ol>

Ejemplo de lista sin orden

<ul>
 <li>Mark</li>
 <li>Jacob</li>
 <li>Craig</li>
</ul>

Ejemplo de lista descriptiva

<dl>
<dt>Nombres</dt>
  <dd>Mark</dd>
  <dd>Jacob</dd>
  <dd>Craig</dd>
</dl>
<dt>Tareas</dt>
  <dd>Limpiar</dd>
  <dd>Lavar trastes</dd>
  <dd>Ordenar</dd>
</dl>
